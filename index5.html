O que significa clonar um repositório e como isso é feito?

Clonar um repositório significa criar uma cópia local completa de um repositório de código fonte. Isso é feito usando o comando git clone [URL] no Git.

Qual é a utilidade de clonar um repositório?

Clonar um repositório permite que você trabalhe com o código localmente, faça alterações e sincronize com o repositório original.
