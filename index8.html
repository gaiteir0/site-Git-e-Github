Como fazer um fork de um repositório?

Para fazer um fork de um repositório, clique no botão "Fork" na página do repositório no GitHub.

O que é um pull request e como criar um pull request?

Um pull request (PR) é uma solicitação para integrar mudanças de uma branch ou fork ao repositório principal. Crie um PR clicando em "Pull Requests" e depois em "New Pull Request" no GitHub.

Como revisar e aceitar um pull request?

Para revisar e aceitar um pull request, analise as mudanças na aba "Files changed", e então clique em "Merge pull request" para aceitar e integrar as mudanças.
