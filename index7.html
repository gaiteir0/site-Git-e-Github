O que são branches e como criar uma nova branch no GitHub?

Branches são ramificações independentes do projeto. Para criar uma nova branch no GitHub, use git branch [nome] e depois git checkout [nome] ou git switch [nome].

Como fazer merge de branches e quais são as boas práticas para isso?

Para fazer merge de branches, use git merge [branch]. Boas práticas incluem revisar e testar alterações antes do merge, resolver conflitos cuidadosamente e manter commits pequenos e claros.
