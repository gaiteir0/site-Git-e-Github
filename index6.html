Quais são os principais comandos do Git que são usados com frequência? (git init, git clone, git add, git commit, git push, git pull)

git init: Inicializa um novo repositório.

git clone [URL]: Clona um repositório existente.

git add [arquivo]: Adiciona arquivos ao estágio de commit.

git commit -m "mensagem": Salva mudanças com uma mensagem.

git push: Envia mudanças para o repositório remoto.

git pull: Atualiza o repositório local com mudanças do remoto

como fazer commits e qual a importância de uma mensagem de commit clara e descritiva?

Para fazer commits, use git commit -m "mensagem". Mensagens claras e descritivas ajudam a entender as mudanças feitas e facilitam a colaboração.
