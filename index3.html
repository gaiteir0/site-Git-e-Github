Como criar uma conta no GitHub?

1. Acesse github.com e clique em “Sign up”.

2. Preencha o formulário com nome de usuário, e-mail e senha.

3. Verifique seu e-mail para confirmar o cadastro.

4. Complete a configuração inicial da conta e explore o perfil.

Quais são os primeiros passos após a criação da conta no GitHub?

1. Configure seu perfil e informações pessoais nas configurações da conta.

2. Crie um repositório: Vá para “New repository” e preencha os detalhes necessários.

3. Explore a interface e familiarize-se com recursos como Issues e Pull Requests.

4. Configure chaves SSH para conexões seguras (opcional, mas recomendado).

Configuração Inicial do Git

Após instalar o Git, é necessário configurar seu nome e e-mail. Utilize os seguintes comandos no terminal:

git config --global user.name "Seu Nome"
git config --global user.email "seu-email@exemplo.com"

Gere e adicione uma chave SSH para conexões seguras:

ssh-keygen -t rsa -b 4096 -C "seu-email@exemplo.com"
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa

Adicione a chave SSH ao GitHub e teste a conexão com:

ssh -T git@github.com
