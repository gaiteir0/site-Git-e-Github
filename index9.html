O que são issues e como abrir uma issue no GitHub?

Issues são itens de rastreamento para bugs, tarefas ou melhorias. Para abrir uma issue no GitHub, vá para a aba "Issues" e clique em "New issue".

Como usar o sistema de projetos do GitHub para gerenciar tarefas?

Use o sistema de projetos do GitHub criando quadros (boards) com colunas e cartões para organizar e gerenciar tarefas e progresso.
